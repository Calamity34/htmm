Before You Ask

Before asking a technical question by E-Mail, in a newsgroup, on a forum or a chat board consider the following first:

  1. Try to find an answer by searching the archives of the forum or the mailing list
  2. Try to find an answer by searching it up in DuckDuckGo/Google
  3. Try to find an answer in the manual/docs
  4. Try to find an answer by reading a FAQ list
  5. Try to find an answer by inspection or experimentation
  6. Try to find an answer by asking a skilled friend if you have one
  7. If you're good at programming - try to find the answer in the source code if avalible

When you ask your question, display the fact that you have done these things first; this will help establish that you're not being a lazy sponge and wasting people's time. Better yet, display what you have learned from doing these things, you will be able to get much more help from demonstrating it.
